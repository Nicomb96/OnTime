Este documento contiene las principales sentencias DDL (Data Definition Language) y DML (Data Manipulation Language) utilizadas en la base de datos de OnTime.

Las DDL definen la estructura de las tablas, mientras que las DML permiten insertar, modificar o eliminar registros. Estas sentencias forman parte esencial del modelo lógico del sistema.
